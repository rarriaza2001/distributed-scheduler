The scheduler’s identity record for a worker process.

WorkerRegistration {
  workerId: string
  hostname: string
  processId: string
  queues: string[]
  concurrency: number
  startedAt: timestamp
  version: string
  status: "active"
}

tells the scheduler:
- this worker exists,
- what it can process,
- how much capacity it has,
- what runtime instance it is,
- what version it is running.

Who writes it
- worker when it boots

Who reads it
- scheduler
- audit tooling

Registration is: identity + capabilities
